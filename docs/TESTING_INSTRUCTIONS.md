## Installation Steps

### Step 1: Enable Unknown Sources (One-Time Only)

1. Open **Settings** on your Android phone
2. Go to **Security** or **Apps** (varies by phone)
3. Find **Install unknown apps** or **Unknown sources**
4. **Enable** it (toggle ON)
5. ✓ You only need to do this once

### Step 2: Download the APK

1. Open the **Google Drive link** we sent you
2. Click the **Download** button (↓ icon)
3. Wait for the download to complete (about 2–3 minutes on WiFi, longer on mobile data)
4. Check your **Downloads** folder when done

### Step 3: Install the App

**Option A: From Files App**
1. Open **Files** or **My Files** app on your phone
2. Navigate to **Downloads**
3. Find **Hanuman_Chalisa_v1.0.0_beta.apk**
4. Tap it → **Install** → **Open**
5. ✓ App should launch automatically

**Option B: From Chrome**
1. If you downloaded via Chrome, Chrome shows a notification
2. Tap **Show downloads** → Find the APK
3. Tap the file → **Install** → **Open**
4. ✓ App should launch automatically

**Option C: Manual Open**
1. Open Files/Downloads
2. Long-press **Hanuman_Chalisa_v1.0.0_beta.apk**
3. Tap **Open with** → Select an app opener
4. Follow prompts to install

## Troubleshooting

### "Cannot find Downloads"
- Try: Open **Files** app → look for **Download** or **Downloads** folder
- Alternative: Open **Chrome** → Menu (⋮) → **Downloads** → Find your file there

### "Installation failed"
- Make sure you enabled **Unknown sources** in Settings first
- Try deleting the file and re-downloading it
- Restart your phone and try again

### "App won't launch"
- Force-close the app: Settings → Apps → Hanuman Chalisa → Force Stop
- Reopen it
- If still broken, uninstall and reinstall

### "Audio cuts off"
- Check your phone's volume (press volume up)
- Try playing a different part of the audio (use the seek bar)
- Restart the app

### "Reminders don't show"
- Check Settings → Apps → Hanuman Chalisa → Notifications → Enable all
- Make sure your phone's notification volume is on
- Try disabling and re-enabling reminders in the app
